# PerguntUP - C/C++
> Um jogo de perguntas e respostas como nenhum outro.


## Features:

- [x] Criar partida;
- [x] Exibir perguntas;
- [x] Responder perguntas;
- [x] SinglePlayer;
- [x] Random ID;
- [x] To Lower;
- [x] Pedir dicas;
- [x] Exibir dicas;
- [x] Dar dicas/recompensas;
- [x] Armazenar dados;
- [x] Print Vector;
- [ ] MultiPlayer;
- [ ] PlayAgain;
