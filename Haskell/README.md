# PerguntUP - C/C++
> Um jogo de perguntas e respostas como nenhum outro.


## Features:

- [ ] Criar partida;
- [ ] Exibir perguntas;
- [ ] Responder perguntas;
- [ ] SinglePlayer;
- [ ] Random ID;
- [ ] To Lower;
- [ ] Pedir dicas;
- [ ] Exibir dicas;
- [ ] Dar dicas/recompensas;
- [ ] Armazenar dados;
- [ ] Print Vector;
- [ ] MultiPlayer;
- [ ] PlayAgain;
- [ ] Exibir Ranking
- [ ] Instruções.
- [ ] Porcentagens "aleatórias"
- [ ] Random Eliminar Respostas
